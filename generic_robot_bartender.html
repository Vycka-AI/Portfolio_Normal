Robotized bar project

Robot bartender project using ABB Robot Studio. Robots were programmed using RAPID programming language. Different drinks can be served and preprogrammed. With a click of a button, order is given to the system. First robot is being used to mix and pour cocktails. Second one serves the drink to a customer.

Models and 3D simulation

In order to have a successful and interactive simulation a few 3D models had to be imported. Those include: robot end effectors, a cup for a drink, cup dispenser, drink itself, ice, soft drink dispensers, aswell as alcoholic beverages themselves and other stuff.

Cocktail structure

There are a few premade cocktails in the project. Every cocktail has it's own unique parameters, such as: how many different ingredients are in it, weather ice is required and such. These parameters can be enumerated with every cocktail, that's why a data structure "RECORD" is used. A record is a composite data type that can hold multiple fields, each with its own data type. For every required liquid a different data structure had to be used, it holds information about: liquid position, amount of liquid needed And wether it's a juice or not.
